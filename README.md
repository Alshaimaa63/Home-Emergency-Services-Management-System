# 🚀 Release Notes: New Version & Enhanced Functionality

## 🌟 Major Updates & Features

### 1. Advanced Rating System
* Implemented **star-based ratings** with automated average calculation.
* Ratings and reviews are now dynamically displayed in the feedback section to build trust.

### 2. Provider Profile Enhancements
* **"About Me" Section:** Providers can now showcase their experience and biography.
* **Multi-skill Selection:** Providers can choose multiple specialties (similar to Khamsat/Fiverr model) to offer diverse services.

### 3. Cross-Profile Accessibility
* **For Customers:** Can now view detailed Provider profiles to make informed decisions.
* **For Providers:** Can access Customer contact details (Address & Phone) once a request is made to facilitate service delivery.

### 4. Identity & Personalization
* **Profile Pictures:** Full support for image uploads for both Customers and Providers across the platform.
* **Customer Dashboard:** Redesigned profile page for easy editing of contact and address information.
